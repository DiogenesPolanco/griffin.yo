Rendering data

Server data (JSON objects) is a first class citizen in Griffin.Yo. Typically the information that you display for the user is generated server side. With Griffin.Yo it takes just one line of code to populate your view with it.

View

<div id="MyView">
    <h2 data-name="Name">
</div>

Code


//from anywhere
Griffin.Yo.G.render('MyView', { Name: 'Arne' });

//from your view model
context.render({ Name: 'Arne' });
