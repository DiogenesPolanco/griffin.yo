Styling list using directies

You can use directives to do styling. For instance if you would like to use zebra stripes.

.

View

<div id="MyView">
    <h2 id="Title"></h2>
    <ul>
        <li data-collection="Items"><span data-name="Name"></span></li>
    </ul>
</div>

Code

var directives = { 
    Items: { 
        Name: { 
            style(value) { return 'font-size: 20pt;'; } 
        }
    }
};
var data = { 
    Title: 'Hello world', 
    Items: [
        { Name: 'Adam'  },
        { Name: 'Bertil' },
        { Name: 'Ceasar' }
    ]
};
Yo.G.render('MyView', data, directives);
