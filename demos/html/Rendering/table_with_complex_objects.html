Rendering a table using complex objects

Griffin.Yo supports collections that consists of objects with deep object graphs.

View

<div id="MyView">
    <h2 id="Title"></h2>
    <table>
        <tr data-collection="Items">
            <td data-name="Name"></td>
            <td data-name="Address">
                <span data-name="Postal"></span>
                <span data-name="City"></span>
            </td>
        </tr>
    </table>
</div>

Code

var data = { 
    Title: 'Hello world', 
    Items: [
        { Name: 'Jonas', Address: { Postal: 12345, City: "Falun" }}, 
        { Name: 'Tom', Address: { Postal: 54321, City: "Stockholm" }}
    ] 
};
Griffin.Yo.G.render('MyView', data);
