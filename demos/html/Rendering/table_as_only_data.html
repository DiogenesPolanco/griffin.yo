Collection as only data

Some libraries have trouble if the data returned from the server is a collection. Griffin.Yo handles that.

View

<div id="MyView">
    <table>
        <tr id="DataTarget">
            <td data-name="Name"></td>
            <td data-name="Address">
                <span data-name="Postal"></span>
                <span data-name="City"></span>
            </td>
        </tr>
    </table>
</div>

Code


Yo.G.render('DataTarget', [
    { Name: 'Jonas', Address: { Postal: 12345, City: "Falun" }}, 
    { Name: 'Tom', Address: { Postal: 54321, City: "Stockholm" }}
]);

Result
