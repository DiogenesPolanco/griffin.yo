Complex object

Complex objects are object graphs that span over more than one level. Griffin.Yo can render complex structures without you having to do anything special. Just create the same structure in your view.

View

<div id="MyView">
    <h2 data-name="FirstName"></h2>
    <div data-name="Address">
        <span data-name="PostalCode"></span>
        <span data-name="City"></span>
    </div>
</div>

Code

var data = { 
    FirstName: "Jonas",
    Address: {
        City: "Falun",
        PostalCode: "791 00"
    }
};            
context.render(data);

Result
