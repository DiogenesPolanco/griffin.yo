Collections

Collections can be rendered to any type of objects. It do not have to be a table or a list, it can be a plain div.

View

<div id="MyView">
    <h2 data-name="Title"></h2>
    <ul>
        <li data-collection="Items"></li>
    </ul>
</div>

Code


var dto = { Title: 'Hello world', Items: ['Adam', 'Bertil', 'Ceasar'] };
Yo.G.render('MyView', dto);
